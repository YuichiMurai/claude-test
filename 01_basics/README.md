# 01_basics - 基礎編

## 学習目標

このセクションでは、Claude APIの基本的な使い方を学びます。

## ファイル一覧

| ファイル | 内容 | 学習時間目安 |
|---------|------|------------|
| 01_api_setup.py | API接続の基本設定 | 20分 |
| 02_first_request.py | 初めてのリクエスト | 20分 |
| 03_streaming.py | ストリーミングレスポンス | 20分 |
| 04_message_history.py | 会話履歴の管理 | 30分 |
| exercises/ | 練習問題 | 30分 |

## 前提条件

- Python 3.8以上
- `pip install -r requirements.txt` 完了済み
- `.env`にAPIキー設定済み

## 実行方法

```bash
python 01_basics/01_api_setup.py
python 01_basics/02_first_request.py
python 01_basics/03_streaming.py
python 01_basics/04_message_history.py
```
