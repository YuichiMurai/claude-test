## 概要

このディレクトリには、Claude APIの基礎を練習するための3つの問題が含まれています。  
各問題は01_basicsで学んだ内容を実践的に応用します。

## 練習問題1: 簡単な質問応答システム (`exercise_01.py`)

### 説明

ユーザーからの質問を受け取り、Claudeで回答を生成するシンプルなQ&Aシステムを実装します。  
`01_api_setup.py` と `02_first_request.py` で学んだ内容を応用します。

### 実装する機能

1. `ask_question(client, question)` 関数：質問を送信して回答を返す
2. `ask_with_context(client, question, context)` 関数：コンテキストを付与して質問する
3. `main()` 関数：複数の質問を試す

### 期待される出力例

```
=== 練習問題1: 質問応答システム ===

質問: Pythonのリスト内包表記とは何ですか？
回答: リスト内包表記は...（Claudeの回答）

質問（コンテキスト付き）: 上記の方法を使って偶数リストを作るには？
回答: ...（Claudeの回答）
```

### ヒント

- `client.messages.create()` を使用してAPIを呼び出す
- システムプロンプトに「簡潔に日本語で回答してください」を設定すると良い
- `response.content[0].text` でテキストを取得できる

## 練習問題2: 会話履歴を使った対話システム (`exercise_02.py`)

### 説明

3ターン以上の会話を実装し、会話のコンテキストを維持する対話システムを構築します。  
`04_message_history.py` で学んだ内容を応用します。

### 実装する機能

1. `SimpleBot` クラス：会話履歴を管理するボット
   - `__init__(client, system_prompt)` メソッド
   - `chat(user_input)` メソッド：1ターンの会話を進める
   - `get_history()` メソッド：会話履歴を返す
   - `reset()` メソッド：会話をリセットする
2. `main()` 関数：3ターン以上の会話を実演

### 期待される出力例

```
=== 練習問題2: 会話履歴を使った対話システム ===

ユーザー: 今日のランチに何を食べればいいですか？
ボット: ...（Claudeの回答）

ユーザー: カロリーが少ないものがいいです
ボット: ...（前の文脈を踏まえた回答）

ユーザー: その中で一番おすすめは？
ボット: ...（さらに文脈を踏まえた回答）

会話ターン数: 3
```

### ヒント

- 会話履歴はリストで管理する（`[{"role": "user", "content": "..."}, ...]`）
- 各ターンでユーザーメッセージを追加し、APIを呼び出す
- APIレスポンスをアシスタントメッセージとして履歴に追加する
- クラスにすることで状態（履歴）を管理しやすくなる

## 練習問題3: ストリーミングを使ったリアルタイム翻訳 (`exercise_03.py`)

### 説明

入力テキストをリアルタイムでストリーミング翻訳するシステムを実装します。  
`03_streaming.py` で学んだ内容を応用します。

### 実装する機能

1. `translate_streaming(client, text, target_language)` 関数：テキストをストリーミング翻訳
2. `translate_multiple(client, texts, target_language)` 関数：複数テキストを順次翻訳
3. `main()` 関数：日本語→英語、英語→日本語の翻訳を実演

### 期待される出力例

```
=== 練習問題3: リアルタイム翻訳システム ===

【日本語 → 英語】
原文: 人工知能は私たちの生活を大きく変えています。
翻訳（リアルタイム）: Artificial intelligence is greatly changing our lives.

【英語 → 日本語】
原文: The quick brown fox jumps over the lazy dog.
翻訳（リアルタイム）: 素早い茶色のキツネが怠け者の犬を飛び越えます。
```

### ヒント

- `client.messages.stream()` を使用してストリーミングを有効化する
- システムプロンプトで「翻訳者」のロールを設定する
- 翻訳以外の説明文が出力されないよう、システムプロンプトで指示する
- `flush=True` を使うとリアルタイム表示ができる
