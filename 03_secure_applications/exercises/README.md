## 🎯 学習のポイント

### これらの練習問題で身につくスキル

1. **セキュリティ意識**: 入力検証とPIIマスキングを「当たり前」にする習慣
2. **堅牢性**: エラーが起きても適切に回復できるコードの書き方
3. **コスト意識**: APIコストを常に意識した設計の考え方
4. **実践力**: 複数の概念を組み合わせた本番品質のコード

### CCA試験との関連

| 練習問題 | 試験での出題ポイント |
|----------|-------------------|
| exercise_01 | プロンプトインジェクション対策、PIIの取り扱い |
| exercise_02 | エラーハンドリング、429エラーの処理、Exponential backoff |
| exercise_03 | トークン管理、コスト最適化、コンテキストウィンドウの管理 |
